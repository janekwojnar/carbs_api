# Roadmap

## Phase 0 (Now: 1-2 weeks)
- Product spec finalization
- Data dictionary and formula registry
- API contracts and audit schema
- UX wireframes and design system

## Phase 1 (MVP: 4-8 weeks)
- Athlete onboarding profile
- Session prediction engine
- Multi-strategy output
- Simulation tool
- PDF/share export
- Garmin + Strava ingest
- Subscription/paywall basics

## Phase 2 (8-16 weeks)
- TrainingPeaks, Zwift, Apple Health
- Improved personalization and explainability
- Race planner with dynamic split fueling
- Stronger analytics and retention loops

## Phase 3 (16-28 weeks)
- Live recommendations in-session
- Advanced uncertainty calibration
- Validation study publication
- iOS companion app using same backend

## Phase 4 (Elite platform)
- Team workflows and advanced dashboards
- More integrations
- Model governance and automated drift monitoring
